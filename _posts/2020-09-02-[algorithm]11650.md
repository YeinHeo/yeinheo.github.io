### [Baekjoon] 11650번 문제 [https://www.acmicpc.net/problem/11650](https://www.acmicpc.net/problem/11650)

```python
n = int(input())
a = []
for i in range(n):
    a.append(list(map(int, input().split())))

b = sorted(a, key=lambda x: x[1])
c = sorted(b, key=lambda x: x[0])

for i in c:
    print(i[0], i[1])
```
